# Caster Camera

The **Caster Camera** is a fully fletched spectator freecam type camera that includes:
- An easy to control freecam mode specifically designed to navigate in arenas
- Seamless Third and First person POV integration with the press of a button
- Driftball 3v3, Driftball 4v4, Driftblitz and Z-Drift support
- Both keyboard and controller support
- Special goal scoring and game start particle effects
- An intutive UI with customizable settings
- Customizable overviews per gamemode
- Fully rebindable keybinds
- Full Overlay support through [Caster Bridge](https://github.com/dennssen/OD_Caster_Bridge)

The Caster Camera is a powerful caster tool for both community and professional casters that allows you to show exactly what you want to show, and when to show it. Partnered with ODC, the Caster Camera is the ultimate tool for any caster looking to cast Orion Drift.

## Dependencies
For POV to work you will need to have installed the newest version of Follow+ which can be found [here](https://github.com/dennssen/Follow-Pluss)

## Manual Installation
Download the file called `dennssen.caster.zip` from the [latest release](https://github.com/dennssen/CasterCamera/releases/latest) and place it in this directory on your computer: `Documents\Another-Axiom\A2\Cameras\Behaviors`. Then extract the `.zip` file in there so that a folder with the name `dennssen.caster` appears.

If you don't see it in the spectator client make sure that when you open the `dennssen.caster` folder it contains files like `main.luau` and `package.json`. If there is another folder inside instead then it won't work.

## Use
You can find information about the Caster Cameras controls by pressing the `?` button at the top of the cameras GUI (F3 menu)

*Note: It is highly recommended to use a controller but keyboard works fine*

*Make sure that none of the keybinds that are being used in this script gets interfered by other scripts keybinds set in the F2 menu if you plan on using a keyboard*
